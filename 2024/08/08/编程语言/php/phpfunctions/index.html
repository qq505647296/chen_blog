语法

PHP 函数准则：

  • 函数的名称应该提示出它的功能
  • 函数名称以字母或下划线开头（不能以数字开头）

实例

一个简单的函数，在其被调用时能输出我的名称：

实例

<?php
function writeName() {
echo “Kai Jim Refsnes”;
}
echo “My name is “;
writeName();
?>

输出：

​ 

My name is Kai Jim Refsnes

​
